# Document Checklist by Visa Type

Standard document lists for the most common visa types. **These are starting points, not gospel.** Re-verify against the destination consulate's own checklist for the user's situation.

## Schengen Tourist Visa (Type C)

Applies to: Italy, France, Germany, Spain, Netherlands, Belgium, Austria, Greece, Portugal, Switzerland, Norway, Sweden, Denmark, Finland, Iceland, Czech Rep., Poland, Hungary, Slovenia, Slovakia, Estonia, Latvia, Lithuania, Luxembourg, Malta, Croatia.

| Item | Specifics |
|---|---|
| Application form | EU Harmonised Schengen Visa Application Form, signed. Many countries also accept their e-application output with 2D barcode. |
| Passport | Original + photocopy of bio page. Valid ≥ 3 months past trip end (some demand 6). ≥ 2 blank visa pages. |
| Photos | 2 biometric photos, 35×45 mm, white background, less than 6 months old, face fills 70–80% of frame, no glasses/hat, neutral expression. |
| Proof of UK residence (or origin country residence) | Residence permit / eVisa share code printout / national ID card. |
| Travel medical insurance | ≥ €30,000 cover, all Schengen states, all trip dates, with insurer name and signature/stamp. |
| Return ticket | Confirmed flight reservation showing name, dates, route, price. |
| Accommodation | Confirmed hotel reservation(s) for every night, OR invitation letter from host with their ID copy. |
| Cover letter | 1 page, signed. Purpose of trip, dates, who's paying, return commitment. |
| Employment letter | If employed: on company letterhead, dated within last 30 days, naming role, salary, approved leave dates, signed. |
| Payslips | Last 3 months. |
| Bank statement | Last 3 months. Show consistent income deposits and adequate balance. Italy expects €269.60 for 1–5 day trip (more for longer stays). |
| Visa fee | €90 adult (subject to change; confirm). Often paid at VFS on the day. |

## US B1/B2 Visitor Visa

| Item | Specifics |
|---|---|
| DS-160 confirmation page | Filled at ceac.state.gov, printed (with barcode). |
| Passport | Valid ≥ 6 months past return. |
| Photo | One 2×2 inch (51×51mm), white background, uploaded to DS-160. Some consulates also want a printed copy. |
| MRV fee receipt | Pay before scheduling. |
| Visa appointment confirmation | OFC + Consular appointment letters. |
| Supporting documents (bring to interview) | Bank statements 3-6 months, employment letter, payslips, property documents, family proofs — anything showing ties to home country (the officer is trying to assess intent to return). |

## UK Standard Visitor Visa

| Item | Specifics |
|---|---|
| Online application | gov.uk/standard-visitor — fill, pay (£127 standard, more for long-stay), book biometrics. |
| Passport | Valid for whole stay. |
| Biometrics | At a UKVCAS / VFS centre. |
| Supporting docs | Upload during application: bank statements 6 months, employment letter, accommodation, travel plans, ties to home country. |

## Japan Tourist Visa (non-visa-exempt nationals)

| Item | Specifics |
|---|---|
| Application form | "Visa application form for temporary visitor". |
| Passport | Valid for entire stay + 6 months recommended. |
| Photo | One 45×45 mm, white background, < 6 months old. |
| Itinerary in Japan | Day-by-day, with accommodation per night. |
| Flight reservation | Return / onward. |
| Bank statements | Last 3 months. |
| Employment letter | If employed. |
| Embassy fees | ~¥3,000 single-entry (varies by reciprocity). |

## Canada Visitor Visa (TRV)

| Item | Specifics |
|---|---|
| Online application | IRCC portal, biometric instructions follow. |
| Passport | Valid for stay duration. |
| Biometrics | At VAC if first time in 10 years. |
| Letter of explanation | Optional but powerful — explain purpose, ties, funding. |
| Financial support | Bank statements 4 months, employment proof, sponsor letter if applicable. |
| Fee | CAD 100 + CAD 85 biometrics. |

## China Tourist Visa (L)

| Item | Specifics |
|---|---|
| Form | Mainland Visa Application Form V.2013, signed. |
| Passport | ≥ 6 months validity, ≥ 1 blank visa page. |
| Photo | 33×48 mm, white background. |
| Hotel + flights | Same as Schengen — full coverage. |
| Invitation letter | Required for some applicants (e.g., from a Chinese citizen or company). |
| Fees | Varies by nationality, paid at CVASC visa centre. |

## Universally useful items (most visas benefit)

Even when not listed by the consulate, the following often resolve officer questions before they're asked:

- **Day-by-day itinerary** — short table: date, city, hotel, key activity.
- **Updated balance screenshot** from the user's bank app on the morning of the appointment, showing the *current* balance (because the statement is by then 3–8 weeks old).
- **Photocopy of any prior visas** in the same passport.
- **Photo of the eVisa / residence status** from the user's foreign-ministry account, if available.

These are not all required, but they preempt the most common follow-up questions.
